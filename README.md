# Calculator Web Application

## Overview
This is a simple and responsive calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations like addition, subtraction, multiplication, and division with a clean user interface.

## Features
- Basic arithmetic operations (+, -, *, /)
- Real-time input and result display
- Clean and simple UI design
- Responsive layout
- Error handling for invalid inputs

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice front-end development skills and understand JavaScript logic, DOM manipulation, and UI design.

## How to Run
1. Download or clone the repository
2. Open the folder
3. Run `index.html` in any browser
